# InvoiceVault Releases

This repository holds the released `.exe` binaries for InvoiceVault.

### How to Add a Release

1. Place your compiled `.exe` file in this folder.
2. Commit and push the changes:
   ```bash
   git add .
   git commit -m "Add new release"
   git push origin main
   ```
3. (Optional but Recommended) Create a new "Release" on the GitHub page (https://github.com/razeltech/InvoiceVault-Releases/releases/new), tag it (e.g., `v1.0.0`), and attach the `.exe` file to the release assets. This provides a direct, clean download URL.
